# Sistem Keuangan Pribadi

Aplikasi PWA untuk mencatat pemasukan & pengeluaran.

## Fitur
- Tambah pemasukan & pengeluaran
- Grafik pengeluaran per kategori
- Pencarian & filter transaksi
- Data tersimpan offline di perangkat
- Bisa diinstall di HP

## Cara Install di HP

### Android (Chrome)
1. Buka website di Chrome
2. Tap menu (3 titik) > Tambahkan ke Layar Utama
3. Tap Install

### iPhone (Safari)
1. Buka website di Safari
2. Tap Share > Add to Home Screen
3. Tap Add

## Cara Buat APK
1. Deploy ke hosting (GitHub Pages / Netlify / Vercel)
2. Buka https://www.pwabuilder.com
3. Input URL > Package > Android > Generate
4. Download APK

## File
- index.html - Halaman utama
- app.js - Logic aplikasi
- sw.js - Service Worker (offline)
- manifest.json - PWA config
- icon-*.png - App icons
